## 1. 프로젝트 개요

### 서비스 소개
실제 OPIc 시험 환경과 동일한 환경에서 모의고사를 제공하고, 키워드 기반 예상 문제 생성 및 커뮤니티 기능을 통해 사용자가 효율적으로 OPIc을 준비할 수 있는 웹 서비스.

### 서비스명
**OPIc Master**

### 디자인 방향
- Toss 디자인 시스템 기반 (색상, 타이포그래피, 컴포넌트)
- 메인 컬러: `#3182f6`
- 폰트: Pretendard
- 아이콘: Iconify (Solar 팩)
- CSS 프레임워크: Tailwind CSS

## 4. 회원 인증

### 4-1. 로그인 / 회원가입
- 카카오 소셜 로그인 연동
- 일반 회원가입 (이메일 + 비밀번호) 선택적 제공, 이메일 회원가입시 인증하지 않음
- 연령, 직업(취준생, 대학생, 대학원생, 직장인), 닉네임 입력 후 최종 회원가입 완료

### 4-2. 계정 유형
| 구분 | 권한 |
|---|---|
| 일반 유저 | 시험 응시, 커뮤니티 게시글 CRUD, 댓글 CRUD, 자료 저장 |
| 어드민 | 일반 유저 권한 + 어드민 전용 기능 전체 |

## 5. 어드민 기능

### 5-1. 대시보드
- 가입자 수 추이 조회 (1일 / 1주 / 1달 기준)
- 차트 형태로 시각화
- 일반 유저 탈퇴 기능 (탈퇴 시 이메일, 닉네임이 모두 임의의 값으로 변경)

### 5-2. 문제 생성 기준 관리
- 현재 오픽 문제 생성에 사용되는 기준 텍스트 조회 및 수정
- 수정 시 이후 생성되는 시험 문제에 실시간 반영
- 기출 문제 데이터 또한 문제 생성 과정에서 반영됨

### 5-3. 기출 문제 관리
- 오픽 실제 출제 문제 리스트 조회
- 문제 추가 / 생성 / 수정 / 삭제 가능

### 5-4. 게시글 관리
- 커뮤니티 전체 게시글 삭제 권한 (오픽 후기 + AI 꿀팁)
- 오픽 자료(유튜브) 게시글 작성 / 수정 / 삭제

### 5-5. 요청사항 관리
- 일반 유저가 제출한 요청 내역 조회
- 답변 작성 및 완료 처리
- 일반 유저 신고 내역 조회 및 경고 처리

## 7. 모의고사


### 7-0. 시험 환경 안내 및 시험 시작 버튼
- 모의고사 페이지 첫 화면에 표시
- 시험 보기 전 유의사항 확인 및 체크박스 선택 시 시험 시작 버튼 활성화

### 7-1. 1차 난이도 선택
- 1단계 ~ 6단계 중 선택

### 7-2. 관심 키워드 선택 (총 12개 이상 필수 선택)
- 상단에 현재 선택한 키워드 개수 안내
- 상단에 인기있는 키워드 12개 목록 표시

#### 직업/학업
| 질문 | 선택지 |
|---|---|
| 현재 종사 분야 | 사업/회사, 재택근무/재택사업, 교사/교육자, 군복무, 일 경험 없음 |
| 현재 학생 여부 | 예, 아니오 |

#### 거주지
- 개인 주택이나 아파트에 홀로 거주
- 친구나 룸메이트와 함께 주택이나 아파트에 거주
- 가족(배우자/자녀/기타 가족)과 함께 주택이나 아파트에 거주
- 학교 기숙사
- 군대 막사

#### 여가 활동 (2개 이상 선택 필수)
영화보기, 해변가기, 당구치기, 요리 관련 프로그램 시청하기, 클럽/나이트클럽 가기, 스포츠 관람, 체스하기, 공연보기, 주거 개선, SNS에 글 올리기, 차로 드라이브하기, 콘서트보기, 술집/바에 가기, 친구들과 문자대화하기, 스파/마사지샵 가기, 박물관가기, 카페/커피전문점에 가기, 시험대비 과정 수강하기, 구지고활동하기, 공원가기, 게임하기, 뉴스를 보거나 듣기, 자원봉사하기, 캠핑하기, TV시청하기, 쇼핑하기, 리얼리티 쇼 시청하기

#### 취미/관심사 (1개 이상 선택)
아이에게 책 읽어주기, 혼자 노래부르거나 합창하기, 그림 그리기, 신문읽기, 음악 감상하기, 춤추기, 요리하기, 여행 관련 잡지나 블로그 읽기, 악기 연주하기, 애완동물 기르기, 독서, 글쓰기(편지/단문/시 등), 주식 투자하기, 사진 촬영하기

#### 운동 (1개 이상 선택)
농구, 골프, 자전거, 하이킹/트레킹, 야구/소프트볼, 배구, 스키/스노우보드, 낚시, 축구, 테니스, 아이스 스케이트, 헬스, 미식축구, 배드민턴, 조깅, 태권도, 하키, 탁구, 걷기, 운동 수업 수강하기, 크리켓, 수영, 요가, 운동을 전혀 하지 않음

#### 휴가/출장 경험 (1개 이상 선택)
국내 출장, 국내 여행, 집에서 보내는 휴가, 해외 출장, 해외여행

### 7-3. 문제 생성
- 목표 레벨 입력
- 1섹션, 2섹션 난이도 선택 (각각 1~6난이도 중 하나 선택 필수)
- 실제 기출 문제 데이터 + 키워드 기반 예상 질문 + 목표 레벨 기반 난이도 고려한 문제 AI 생성

### 7-4. 결과 페이지 액션 버튼

| 버튼 | 동작 |
|---|---|
| 시험 문제만 다운받기 | 문제 텍스트 파일 다운로드 |
| 시험 시작하기 | 준비 확인 알럿 → 예: 시험 페이지 이동 / 아니오: 기존 화면 유지 |

**시작 알럿 문구:**
> "시험의 제한시간은 40분이며, 도중에 멈출 수 없으니 주변 환경 체크해주세요. 준비 되셨나요?"
> [예] [아니오]

## 8. 모의고사 진행 흐름

```
모의고사 페이지 진입
    └─ [시작하기] 버튼 클릭
           └─ 1차 세션 시작 (자기소개 포함 총 7문제)
                  └─ 2차 세션 (난이도에 따라 문제 수 결정)
                         └─ 난이도 1~2: 5문제 / 난이도 3~6: 8문제
                                └─ 전체 완료 OR 40분 경과 → 시험 종료
                                       └─ 시험 결과(시험지) 페이지
```

### 8-1. 시험 규칙
- 총 제한 시간: 40분
- 전체 문제 수: 자기소개 포함 12문제(2세션 난이도 1~2) 또는 15문제(2세션 난이도 3~6)
- 문제별 제한 시간 없음, 40분 내 전체 완료
- 시험 도중 일시정지 불가, '나가기' 선택 시 저장되지 않음 주의 알럿 제공(그래도 나가시겠습니까? 예/아니오), 예 선택 시 모의고사 목록 페이지로 이동

### 8-2. 시험 종료 후 - 시험지 조회

| 항목 | 내용 |
|---|---|
| 사용자가 선택한 난이도 | 조회 |
| 키워드 선택 내역 | 조회 |
| 출제 질문 목록 | 조회 |
| 답한 내용 다시 듣기 | 질문별 녹음 재생 |
| 답한 시간 | 질문별 소요 시간 조회 |
| 시험지 다운로드 | 다운로드 버튼 |
| 시험 다시 보기 | 동일 문제로 재응시 버튼 |

## 11. 마이페이지

### 11-1. 요청사항
- **요청하기**: 관리자에게 문의 글 작성
- **요청 내역**: 제출한 요청 목록 조회
  - 답변 완료 시 완료 표시
  - 클릭 시 답변 팝업 형태로 확인

### 11-2. 본 시험 내역
- 사이트에서 응시한 시험 기록 전체 조회
- 각 시험지로 이동 가능

### 11-3. 저장한 글
- 커뮤니티 및 오픽 자료에서 저장한 글 모아보기
- 게시판 구분 없이 저장 글 통합 조회

### 11-4. 계정
- 로그아웃
- 회원 탈퇴 - 탈퇴 사유 입력란 필요
- 회원 탈퇴시 즉각 반영(DB에서 즉시 삭제)
