> 최종 수정일: 2026-06-03

## 1. 프로젝트 개요

### 서비스 소개
키워드 기반 AI 예상 문제 생성을 핵심으로, 사용자가 자신의 배경과 목표 레벨에 맞는 OPIc 예상 문제를 즉시 뽑아 공부할 수 있는 웹 서비스. 커뮤니티 기능을 통해 수험 경험을 나누고, 유튜브 자료 아카이브로 학습 리소스를 한 곳에서 관리한다.

### 서비스명
**OPIc Master**

### 핵심 가치
- **생성**: 내 키워드 + 목표 레벨 기반 AI 문제 생성
- **저장**: 생성한 문제 세트를 저장하고 다운로드
- **공유**: 커뮤니티에서 후기·공부법 교류

### 디자인 방향
- Toss 디자인 시스템 기반 (색상, 타이포그래피, 컴포넌트)
- 메인 컬러: `#3182f6`
- 폰트: Pretendard
- 아이콘: Iconify (Solar 팩)
- CSS 프레임워크: Tailwind CSS

## 4. 회원 인증

### 4-1. 로그인 / 회원가입
- 카카오 소셜 로그인 연동
- 일반 회원가입 (이메일 + 비밀번호), 이메일 인증 없음
- 연령, 직업(취준생, 대학생, 대학원생, 직장인), 닉네임 입력 후 최종 가입 완료

### 4-2. 계정 유형
| 구분 | 권한 |
|---|---|
| 일반 유저 | 문제 생성, 문제 세트 저장/다운로드, 커뮤니티 게시글 CRUD, 댓글 CRUD, 자료 저장 |
| 어드민 | 일반 유저 권한 + 어드민 전용 기능 전체 |

## 5. 어드민 기능

### 5-1. 대시보드
- 가입자 수 추이 조회 (1일 / 1주 / 1달 기준)
- 차트 형태로 시각화
- 일반 유저 탈퇴 처리 (탈퇴 시 이메일·닉네임을 임의값으로 변경)

### 5-2. 문제 생성 기준 관리
- 현재 AI 문제 생성에 사용되는 기준 텍스트 조회 및 수정
- 수정 즉시 이후 생성되는 모든 문제에 반영

### 5-3. 기출 문제 관리
- 오픽 실제 기출 문제 리스트 조회
- 문제 추가 / 수정 / 삭제 가능
- 기출 문제 데이터는 AI 문제 생성 시 참조됨

### 5-4. 게시글 관리
- 커뮤니티 전체 게시글 삭제 권한 (오픽 후기 + 공부법 후기)
- 오픽 자료(유튜브) 게시글 작성 / 수정 / 삭제

### 5-5. 요청사항 관리
- 일반 유저가 제출한 요청 내역 조회
- 답변 작성 및 완료 처리
- 신고 내역 조회 및 경고 처리

## 7. 문제 생성 (핵심 기능)

### 7-1. 1단계 — 난이도 선택
- 1단계 ~ 6단계 중 선택

### 7-2. 2단계 — 관심 키워드 선택 (총 12개 이상 필수)
- 상단에 현재 선택 키워드 수 실시간 표시
- 인기 키워드 12개 상단 노출

#### 직업/학업
| 질문 | 선택지 |
|---|---|
| 현재 종사 분야 | 사업/회사, 재택근무/재택사업, 교사/교육자, 군복무, 일 경험 없음 |
| 현재 학생 여부 | 예, 아니오 |

#### 거주지
- 개인 주택이나 아파트에 홀로 거주
- 친구나 룸메이트와 함께 주택이나 아파트에 거주
- 가족(배우자/자녀/기타 가족)과 함께 주택이나 아파트에 거주
- 학교 기숙사
- 군대 막사

#### 여가 활동 (2개 이상 선택 필수)
영화보기, 해변가기, 당구치기, 요리 관련 프로그램 시청하기, 클럽/나이트클럽 가기, 스포츠 관람, 체스하기, 공연보기, 주거 개선, SNS에 글 올리기, 차로 드라이브하기, 콘서트보기, 술집/바에 가기, 친구들과 문자대화하기, 스파/마사지샵 가기, 박물관가기, 카페/커피전문점에 가기, 시험대비 과정 수강하기, 구직 활동하기, 공원가기, 게임하기, 뉴스를 보거나 듣기, 자원봉사하기, 캠핑하기, TV시청하기, 쇼핑하기, 리얼리티 쇼 시청하기

#### 취미/관심사 (1개 이상 선택)
아이에게 책 읽어주기, 혼자 노래부르거나 합창하기, 그림 그리기, 신문읽기, 음악 감상하기, 춤추기, 요리하기, 여행 관련 잡지나 블로그 읽기, 악기 연주하기, 애완동물 기르기, 독서, 글쓰기(편지/단문/시 등), 주식 투자하기, 사진 촬영하기

#### 운동 (1개 이상 선택)
농구, 골프, 자전거, 하이킹/트레킹, 야구/소프트볼, 배구, 스키/스노우보드, 낚시, 축구, 테니스, 아이스 스케이트, 헬스, 미식축구, 배드민턴, 조깅, 태권도, 하키, 탁구, 걷기, 운동 수업 수강하기, 크리켓, 수영, 요가, 운동을 전혀 하지 않음

#### 휴가/출장 경험 (1개 이상 선택)
국내 출장, 국내 여행, 집에서 보내는 휴가, 해외 출장, 해외여행

### 7-3. 3단계 — 목표 레벨 및 섹션 난이도 설정
- 목표 레벨 선택 (IL / IM1 / IM2 / IM3 / IH / AL)
- 1섹션, 2섹션 난이도 각각 선택 (1~6단계)

### 7-4. AI 문제 생성
- 실제 기출 문제 데이터 + 선택 키워드 + 목표 레벨 기반으로 AI가 문제 생성
- 생성 중 로딩 인디케이터 표시
- 생성 완료 후 결과 페이지로 자동 이동

### 7-5. 결과 페이지

| 표시 항목 | 내용 |
|---|---|
| 선택한 난이도 | 조회 |
| 선택한 키워드 목록 | 조회 |
| 목표 레벨 | 조회 |
| 생성된 문제 목록 | 섹션별 구분하여 전체 표시 |

#### 액션 버튼
| 버튼 | 동작 |
|---|---|
| 문제 세트 저장하기 | 마이페이지 저장 목록에 추가 (로그인 필요) |
| 문제 다운로드 | 문제 텍스트 파일 (.txt) 다운로드 |
| 다시 생성하기 | 동일 조건으로 새 문제 재생성 |
| 새로 만들기 | 문제 생성 1단계로 이동 |

## 13. 데이터 모델 (주요 변경)

기존 `Exam` / `ExamAnswer` 모델은 문제 생성 결과 저장 모델로 재설계.

| 모델 | 역할 |
|---|---|
| `GeneratedSet` | 생성된 문제 세트 (userId, difficulty, targetLevel, keywords, questions JSON) |
| `Question` | 기출 문제 저장 (AI 생성 시 참조) |
| `ExamGenerationCriteria` | AI 생성 기준 텍스트 |
| `Post` / `Comment` | 커뮤니티 게시글·댓글 |
| `Resource` | 유튜브 자료 |
| `SavedItem` | 유저가 저장한 문제 세트·게시글·자료 |
| `Request` / `Report` | 요청사항·신고 |

> 기존 `ExamAnswer` (녹음 파일 관련) 및 AWS S3 음성 저장 기능 제거.
